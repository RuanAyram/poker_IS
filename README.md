# Simple Poker Game

## Getting started

Clone the repository
```
git clone git@github.com:RuanAyram/poker_IS.git
```

Enter in the folder

```
cd poker_IS
```

## Then, install the rspec gem:

`gem install rspec`

## For example test, run:

```
cd examples/
ruby simple_game.rb
```

## For tests, run in root:

`rspec`
